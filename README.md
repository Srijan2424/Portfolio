# 🌐 Portfolio Website (React)

## 📌 Overview
This is a personal portfolio website built using React and React Router.  
It showcases my profile, skills, research interests, and projects.

The website is designed with a modern UI and deployed using GitHub Pages.

---

## 🚀 Features
- Home Page:
  - About Me
  - Research Interests
  - Personal Details
  - Profile Image
- Projects Page:
  - Displays projects
  - Links to GitHub repositories
- Navigation using React Router (NavLink)
- Clean and modern styling using CSS

---

- HTML5
- CSS3
- JavaScript

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/your-username/portfolio.git

2. Navigate to the project:
cd portfolio

3. Install dependencies:
npm install

4. Run the app locally:
npm start

---

## 🌍 Deployment (GitHub Pages)

1. Install gh-pages:
npm install gh-pages --save-dev

2. Add this to package.json:
"homepage": "https://your-username.github.io/portfolio"

3. Add scripts:
"predeploy": "npm run build",
"deploy": "gh-pages -d build"

4. Deploy the project:
npm run deploy

5. Enable GitHub Pages:
- Go to Repository → Settings → Pages
- Select branch: gh-pages
